---
description: How to deploy changes to GitHub
---

Follow these three steps to push your latest changes to GitHub:

1.  **Stage your changes**:
    ```bash
    git add .
    ```

2.  **Commit with a message**:
    ```bash
    git commit -m "Describe what you changed"
    ```

3.  **Push to GitHub**:
    ```bash
    git push
    ```
